<h3> A Word Guess game in python </h3>

<ul>
    <li> The textfile names anat19.txt contains the text which will be used. </li>
    <li> The text is tokenized, lemmatized and tagged according to their type (Verb, noun, adjective and such). </li>
    <li> Nouns are extracted from the tagged list and stored in descending order of frequency (highest first). </li>
    <li> Function 'word_guess' contains the code for the game which uses the noun list. </li>
    <li> Players are given 5 points in the beginning, which increments and decrements according to right and wrong guesses. </li>
</ul>

<h4> This game was created as an assignment for the CS6320 (Natural Language Processing) course of UTD. </h4>
